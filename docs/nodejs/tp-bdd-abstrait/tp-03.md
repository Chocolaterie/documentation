# TP Node.js BDD - 03

## Contexte

Ce TP est un bonus et demande davantage d'autonomie.

Vous allez ajouter **SonarQube** à votre projet afin d'analyser la qualité de votre code.

## Fonctionnalités / Améliorations

Pour aller plus loin, vous pouvez ajouter :

- L'installation de **SonarQube en local**
- La configuration de Sonar dans votre projet Express
- La configuration de **Jest** pour inclure uniquement les *services* dans le code coverage

## Remarque

Ce TP peut être plus complexe que les précédents, car il nécessite de vous documenter en autonomie, notamment si c'est votre première utilisation de SonarQube.

Je peux vous donner quelques indications, notamment via certaines de mes synthèses pour installer SonarQube en local sur votre machine :

[Installer SonarQube](/docs/general/sonar)
