# TP Spring Web - Partie 05 (p3)

> **Durée Estimée : 1-2h**

## Objectif

- Faire les modifications nécessaires pour, à présent, utiliser un objet dans le formulaire de connexion.
- Il faut donc envoyer un `User` (AppUser pour moi) dans le formulaire.
- Récupérer le `User` via la requête POST (traitement du formulaire).

### Point de départ : la correction

Si vous partez de la correction, il faut modifier les URLs existantes, à savoir :
- `showLoginForm`
- `loginProcess`

C'est dans `loginProcess` qu'on devra :
- être en **POST** ;
- récupérer le `User` avec `@ModelAttribute` ;

Car on appellera le service de login via le `User` provenant du formulaire, et non avec un utilisateur **codé en dur**.
