# TP 05 - Implementer JPA 

Suite TP Suivis Repas

> **Note:** Avant de démarrer ce TP, il convient d’avoir suivi les vidéos des modules 1 à 5 et d’avoir réalisé les TP proposés.

**Durée Estimée : 1h**

## Enonce

En vous basant sur la Demo du module 5, vous allez devoir implementer la version JPA de votre TP.

Donc créer une class **DAOMealImplJPA** qui va implémenter l'interface **DAOMeal**

Et donc dans la classe **DAOFactory** il sera possible de switcher entre **DAOMealImpl** et **DAOMealImplJPA** dans la méthode **getDAOMeal**
